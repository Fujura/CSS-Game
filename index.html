Свойство position: absolute

Устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице.

Рассмотрим следующее изоображение:

Когда мы создаем некий дочерний элемент в родительский блок, то (без никаких дополнительных свойств) данный объект всегда распологается в левом верхнем углу.

*Если в блоке содержится один или несколько элементов, то данный блок называется родительским.

*Если элемент принадлежин какому либо блоку, он называется дочерним.

Когда мы указываем наше свойство, то с первого взгляда ничего не поменялось, однако мы установили для нашего дочернего элемента невидимую обратную координатную плоскость, в которой блок стоит по умолчанию на координатах Y:0 и X:0

.cube{

background-color: red;

width: 100px;

height: 100px;

position: absolute;

}

Однако для управлением координатными точками требуется вспомогательное свойство top и left

*Обратная координатная плоскость, в котором положительный коэффицент Y направлен вниз

Свойства top и left

*Они регулируют местоположение, в котором будет находиться элемент

Свойство top отвечает за вертикальное положение объекта:

.cube{

background-color: red;

width: 100px;

height: 100px;

position: absolute;

top: 15px;

}

Свойство left отвечает за горизонтальное положение объекта:

.cube{

background-color: red;

width: 100px;

height: 100px;

position: absolute;

top: 15px;

left: 30px;

}

Свойство transform

Трансформирует блок в другом виде или месте. Часто применяется со свойством transition.

Виды трансформации

translateX(100px) - перемещает объект по оси Х

translateY(100px) - перемещает объект по оси Y

scaleX(2) - увеличивает объект по оси X

scaleY(2) - увеличивает объект по оси Y

rotate(180deg) - поворачивает объект на заданное количество градусов

skewX(30deg) - задает наклон элемента по оси X

skewY(30deg) - задает наклон элемента по оси Y

Свойсто display: flex

Обычно, когда мы создаем несколько элементов в родительский блок, то они стоят на левом крае в ряд с вверху вниз.

Однако, если мы в родительский блок запишем свойство display: flex, то дочерние элементы встанут в ряд

#parent{

border: 3px solid black;

background-color: #efe4b0;

width: 150px;

height: 100px;

display: flex;

}

Многие знают про вспомогательное свойство justify-content, однако мало кто знает про еще два свойства: flex-direction и align-items
